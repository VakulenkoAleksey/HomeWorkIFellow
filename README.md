# HomeWorkIFellow
Задача: Найти валидные круглые скобки. 
Обязательно счетчик и правильные скобки нужно выводить.
 
 
Пример 1:
               Ввод: "(()"
               Вывод: 2 - ()
Пример 2:
               Ввод: ")()())"
               Вывод: 4 - ()()
Пример 3:
               Ввод: ")(()())"
               Вывод: 6 - (()())
Пример 4:
               Ввод: ")("
               Вывод: 0
Пример 5:
               Ввод: "())(()())(()"
               Вывод: 10 - ()(()())()
